# LABORATORIO 001
Descripción:Creación de primer programa para hacer un blink, es decir hacer parpadear una luz
## COMPONENTES
* 1 Led
* 2 Cables Jumper
* 1 Resistencia de 220 ohm
* 1 Breadboard (es la placa blanca)
* 1 Arduino ONE

ARDUINO (pin 8) -> Cable rojp -> Breadboard -> (+) Resistencia (-) -> Led -> ARDUINO (GRD)
